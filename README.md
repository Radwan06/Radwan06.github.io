SITE WEB PORTFOLIO

Ce projet est un site web portfolio académique réalisé dans le cadre du cours de SEG. Il présente mon profil, mon expérience débutante en développement d’interface utilisateur, mes sources d’inspiration et mes choix de design. Le site a été conçu pour être simple, lisible, professionnel et responsive.

Technologies utilisées : HTML5, CSS3, Bootstrap 5.

Choix de design : palette de couleurs sobre, typographie sans serif pour une bonne lisibilité et disposition basée sur la grille Bootstrap.

Inspiration : Awwwards (minimalisme et clarté visuelle).

Hébergement : GitHub Pages.
