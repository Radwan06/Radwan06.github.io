SITE WEB PORTFOLIO 

Ce projet est un site web portfolio académique réalisé dans le cadre du cours de SEG. Dans ce site web, je présente mon profil, mon expérience débutante en UI, mes sources d’inspiration, ainsi que mes choix de design en lien avec la communication visuelle. Le site a été conçu pour être simple, lisible et professionnel.

Technologies utilisées

HTML5

CSS3

Bootstrap 5 (via CDN)

Fonctionnalités

Mise en page responsive (ordinateur et mobile)

Navigation claire et structurée

Sections dédiées à l’expérience, à l’inspiration, au projet et au code source

Hébergement compatible avec GitHub Pages

Choix de design

Couleurs : palette sobre afin de transmettre un aspect professionnel et académique

Typographie : police sans serif pour une meilleure lisibilité

Disposition : utilisation de la grille Bootstrap pour une structure claire et cohérente

Inspiration

Le design s’inspire de sites comme Awwwards, notamment pour le minimalisme, la hiérarchie visuelle et la clarté de la présentation.

Hébergement

Le site est hébergé sur GitHub Pages et accessible via le lien fourni dans le dépôt.
